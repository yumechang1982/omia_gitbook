# 如何成為OMIA會員？

請您點選OMIA首頁右上角「登入」按鈕→頁面下方「還不是會員？」點擊［註冊］按鈕開始註冊。

<figure><img src="../../.gitbook/assets/註冊會員_4.png" alt=""><figcaption></figcaption></figure>

您可使用Facebook、LINE或Google帳號快速註冊登入，亦可使用下方的「手動輸入電子信箱帳號／密碼方式」註冊成為OMIA新會員。

<figure><img src="../../.gitbook/assets/快速登入_5.png" alt=""><figcaption></figcaption></figure>

**【小提醒】**&#x82E5;使&#x7528;**「手動輸入電子信箱方式」**&#x8A3B;冊，需完成信箱驗證。註冊後請至信箱收取「會員認證信」。
